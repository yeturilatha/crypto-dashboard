This project is a responsive cryptocurrency dashboard built using React and the CoinGecko API.

## Features
- Cryptocurrency selection (Bitcoin, Ethereum, Solana)
- Historical price chart (7, 14, 30 days)
- Trading volume chart
- Top gainer and top loser (24h)
- Centralized state management using React Context API
- Responsive UI using Material UI

## Tech Stack
- React
- Material UI
- Chart.js
- Axios
- CoinGecko API

## API Used
- /coins/{id}/market_chart
- /coins/markets

## Setup Instructions

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000
